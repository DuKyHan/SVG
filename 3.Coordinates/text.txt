#1/The area of the canvas your document intends to use is called the viewport
 You establish the size of this viewport with the width and height attributes on the <svg> element.
 <svg width="200px" height="150px">
 Both of these specify an area 200 pixels wide and 150 pixels tall.
#2/ Using Default User Coordinates
